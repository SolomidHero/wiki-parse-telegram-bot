# Wiki parser bot
based on Telegram by python3

### Install

In git repo.
Activate virtualenv

    source ./bin/activate

Required packages

    pip3 install python-telegram-bot --update
    pip3 install beautifulsoup4
    pip3 install NumPy
    pip3 install matplotlib
    pip3 install wordcloud

Logout

    deactivate

### Setup

    python3 main.py

### Under proxy

    pip3 install PySocks
